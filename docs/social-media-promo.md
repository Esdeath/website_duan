去年开始，我决定自己把这些内容整理成一个能反复回看的知识库。今年终于做完了一个相对完整的版本——**《段永平投资问答录｜段永平投资问答录》**：91 篇文章、5 大板块。

📚 **内容结构**

- **核心概念**（42 篇）— 投资理念、企业经营、核心哲学、品格与心性、财务指标五个子方向
- **访谈实录**（28 篇）— 从 2000 年央视演讲到 2010 年北大对谈，按时间线归档
- **投资问答录**（6 章）— 商业模式、公司点评、人生箴言等专题
- **投资逻辑**（7 篇）/ **商业逻辑**（8 篇）— 抽出来的纯方法论

🛠 **我是怎么做的**

1. **手工逐篇收集** —— 从书籍、雪球、博客一篇一篇找回来，对照原文校对，剔除重复
2. **AI 辅助整理** —— 用 Claude 协助生成摘要、做主题归类、写元数据，但每一篇都自己审一遍才放上去
3. **重新组织成主题** —— 不是简单堆料，而是按"概念—访谈—问答—逻辑"四层重新搭骨架，让人能按需取用
4. **网站和电子书都是自己写的** —— Nuxt 4 + Vue 3 搭的静态站，自己写了一个脚本能把全部内容打包成 EPUB 和 PDF，方便离线阅读

🌗 **使用体验**

- 分类侧边栏，按主题快速跳转
- 每篇文章右侧有目录，长文不迷路
