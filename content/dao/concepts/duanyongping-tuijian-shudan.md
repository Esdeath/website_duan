# 段永平推荐书单

**段永平**很少完整看完一本书，多数书他只是“翻一翻”。他一再强调：

> 已经有这么多好书了，我干嘛要添乱。——2010-05-05
>
> 我还是认为对于投资关键在悟，看书太多没太大意思，反而浪费时间。——2011-06-30
>
> 好书一本就够，有空多看看也无妨。——2010-05-05

所以这份清单的重点不在“多”，而在区分：哪些是他**本人**真正读过并推荐的，哪些只是被网友提到、他其实没看过，哪些是他**明确反对**的。每条都注明出处时间，便于核对。

## 一、他亲口推荐的书

最权威的一条来自 2010-07-08，网友请他开个书单，他回答：

> 排名不分先后：巴菲特给股东的信、巴菲特名言录、基业长青、韦尔奇自传。有空还可以看：查理宝典、穷爸爸富爸爸。——2010-07-08

综合各处反复推荐的，核心书目如下：

| 书名 | 作者 | 他的态度（出处） |
| --- | --- | --- |
| **《巴菲特致股东的信》** | 沃伦·巴菲特 | 头号推荐。“本人唯一看过的书”“最重要的书是巴菲特写给股东的信”（2010-02-09 / 2010-02-11） |
| **《因为独特》** | 李翔 | 极力推荐。“大学毕业后第一本读完了的书”，反复让人“先好好看看这本书”（2026-04-06 / 2026-04-11） |
| **《基业长青》**（Built to Last） | 吉姆·柯林斯 | “是本好书啊”“确实值得看”，印象最深的是 stop-doing list（2012-03-25 / 2013-04-02） |
| **《杰克·韦尔奇自传》** | 杰克·韦尔奇 | 对他建立企业文化“印象最深”，也是巴菲特向股东推荐的书（多处） |
| **《巴菲特名言录 / 平时言论》** | — | 排在《致股东的信》之后，“其次是巴菲特平时的言论”（2010-02-09） |
| **《穷查理宝典》** | 整理自查理·芒格（李录作序） | “这本书不错”“在此再次推荐一下”，称赞李录的序（2010-05-04 起多处） |
| **《富爸爸，穷爸爸》** | 罗伯特·清崎 | 给初学者：“觉得巴菲特一时不好懂，可以看看这本书”（2010-05-26 等） |

延伸阅读（他也多次正面提到）：

- **《赢（Winning）》**——杰克·韦尔奇。讲“[企业文化](/qiyewenhua)如何传递”，多次“建议看韦尔奇的《赢》”。
- **《从优秀到卓越》（Good to Great）**——吉姆·柯林斯。常和《基业长青》一起推荐：“建议看下《基业长青》和《从优秀到卓越》这两本书”（2019-03-23）。
- **《巴菲特忠告中国股民》**——严行方。专门给[价值投资](/jiazhitouzi)初学者推荐：“作者对巴菲特的理解非常好……值得一看”（2011-01-04）。

## 二、他读过并称赞的书

- **《乔布斯传》**——沃尔特·艾萨克森。最后一章“遗产：无比辉煌的创新天堂”里乔布斯的那段自述，他说“**我反复读了很多遍**，字里行间跃然纸上的是一个虽有阴暗面却始终闪亮的灵魂”。

## 三、常被误以为他推荐，其实他没读过 / 中立

- **《聪明的投资者》**（格雷厄姆）——他本人说：“**我没看过**，但我知道里面说啥，其实就一句话：买股票就是买公司。”（2012-05-08）需注意：访谈实录中大段盛赞《聪明投资者》第 8、20 章的，是**巴菲特的原话**，不是段永平。
- **《道德经》**——被问过两次，他都明确回答“**没看过 / 没读过**”（2025-11-03 / 2026-01-27）。
- **《证券分析》**——“我没看过《证券分析》”（2011-01-27）。
- **《滚雪球》**——“看了几页，看不下去……写得琐碎，而且我发现巴菲特不喜欢这本书。”（2010-03-19）不了解巴菲特的人可以看看，但他本人不推荐。

## 四、他明确不推荐 / 反对的书

- **欧奈尔的技术分析系列**（《笑傲股市》《股票投资的24堂必修课》《股票买卖原则》《如何在卖空中获利》）——“**投资看这些书很容易亏钱的。**”（2010-03-30）
- **《趋势跟踪》及网传“十大投资书籍”那一类**——“一本都没看过……**这种书看多了对投资一定是帮倒忙的。**”
- **山寨书**《中国股神学炒股》《段永平的80条炒股妙招》等——与他无关，他从未写书或授权他人写书。

## 小结

段永平真正反复推崇的，是**《巴菲特致股东的信》**——“看懂了巴菲特，别的书不看也没关系”。其余如《因为独特》《基业长青》《杰克·韦尔奇自传》《穷查理宝典》《富爸爸，穷爸爸》等，是延伸阅读。理解了他“关键在悟、好书一本就够”的态度，比把书单读完更重要。
