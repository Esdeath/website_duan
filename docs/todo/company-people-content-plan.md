## 泡泡玛特专题写法

- [x] 标题建议：泡泡玛特：段永平如何从“看不懂”到重新评估这家公司
- [x] 写作定位：新近观察 / 重估案例
- [x] 不要直接写成“段永平已经完全看懂并无条件看好”
- [x] 必须写清楚早期保留：产品和创始人有意思、公司很了不起，但看不懂 10 年后
- [x] 必须写清楚后续重估：情绪价值、用户关注度、艺术家签约、全球门店、王宁和团队
- [x] 必须写清楚核心风险：潮玩需求的持续性、IP 生命周期、爆品依赖、用户热情波动
- [x] 推荐标签：情绪价值、品牌、IP、全球化、商业模式、重估案例

## 王宁专题写法

- [x] 标题建议：王宁：段永平为什么欣赏泡泡玛特创始人
- [x] 写作定位：人物观察，不写人物传记
- [x] 核心结论：段永平欣赏王宁，重点在商业理解、长期努力、做对事情意识、细节能力和团队能力
- [x] 必须写清楚：欣赏王宁不等于泡泡玛特没有风险

- [x] 推荐标签：创始人、CEO、做对的事情、商业理解、细节、长期主义

## 内容原则

- [x] 每篇都要有“原文依据 + 归纳 + 保留条件”
- [x] 明确区分“高度评价”“正面提及”“曾高度评价但后来修正”“正在重估”
- [x] 原文没有直接点名的人，不要写成段永平原文点名
- [x] 对泡泡玛特和王宁，优先写“观察变化过程”，不要写成盖棺定论
- [x] 对阿里、GE、网易、腾讯、英伟达等有保留的对象，保留部分必须和正面评价放在同一篇里
- [x] 所有文章尽量链接到已有概念页：商业模式、企业文化、护城河、消费者导向、本分、做对的事情、长期主义、能力圈、平常心
